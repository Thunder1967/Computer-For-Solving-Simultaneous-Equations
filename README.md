# 計算機網址 : https://thunder666.w3spaces.com/

## 介紹
一個用高斯消去解聯立方程式的計算機
- 支援最簡分數輸出(用來寫作業、對答案)
- 自創暴力計算法 😆
### 使用方法:
#### 1. 選擇計算方式
- Int模式: 輸入會被無條件捨去；計算過程中不使用除法且不產生小數
- float模式: 計算過程中會產生小數，有一定的誤差
#### 2. 輸入聯立方程式(矩陣)係數
- 每行以空白間隔；每列以換行間隔，輸入聯立方程式(矩陣)係數
- 輸入中的非數字會被自動忽略
- 此運算程式即用高斯消去求解
#### 3. 執行並查看運算結果
- 可至進階設定調整輸出方式
